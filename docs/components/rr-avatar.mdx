## Properties

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `md` | Circle diameter. |
| `color` | `'default' \| 'indigo' \| 'sky' \| 'green' \| 'amber'` | `default` | Accent-bold color variant for the initials surface. |
| `name` | `string` | `` | Full name used to derive up to two initials and the accessible label. |
| `src` | `string` | `` | Image URL. When set, an img fills the circle instead of initials. |
| `alt` | `string` | `` | Alt text for the image; falls back to name when omitted. |

## Accessibility

- **WAI-ARIA pattern:** [www.w3.org/WAI/ARIA/apg/patterns/](https://www.w3.org/WAI/ARIA/apg/patterns/)
- **Focusable:** no
- **WCAG:** 1.1.1 Non-text Content, 1.4.3 Contrast (Minimum), 1.4.11 Non-text Contrast

**Rules**

- Initials avatars expose role=img with aria-label set to name; the initials text itself is aria-hidden
- Image avatars derive alt from the alt prop, falling back to name
