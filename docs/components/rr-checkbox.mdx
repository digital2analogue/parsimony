## Overview

Use `rr-checkbox` for independent yes/no choices where several can be on at once — settings lists, multi-select filters, terms acceptance. For mutually exclusive options use `rr-radio-group`; for an instant-effect on/off switch use `rr-toggle` (a checkbox reads as "part of a form, applies on submit"; a toggle reads as "applies now"). It's form-associated via ElementInternals, so `name`/`value` participate in native form submission. `indeterminate` is a display state for "some children checked" in tree patterns — it never submits; clicking resolves it to checked.
