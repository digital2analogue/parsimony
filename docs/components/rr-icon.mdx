## Examples

### Decorative icon (screen readers skip it)

```html
<rr-icon size="default">
  <svg viewBox="0 0 24 24"><circle cx="12" cy="12" r="10" /></svg>
</rr-icon>
```

### Labelled standalone icon

```html
<rr-icon aria-label="Close dialog">
  <svg viewBox="0 0 16 16"><path d="M4 4L12 12M12 4L4 12" /></svg>
</rr-icon>
```

### Large size

## Accessibility

- **WAI-ARIA pattern:** [www.w3.org/WAI/ARIA/apg/](https://www.w3.org/WAI/ARIA/apg/)
- **Focusable:** no
- **WCAG:** 1.1.1 Non-text Content, 4.1.2 Name, Role, Value

**Rules**

- Omit aria-label for purely decorative icons — the component applies aria-hidden="true" automatically
- Set aria-label only when the icon conveys meaning not present in adjacent text — it then gets role="img"
