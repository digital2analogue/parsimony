## Accessibility

- **WAI-ARIA pattern:** [www.w3.org/WAI/ARIA/apg/patterns/textbox/](https://www.w3.org/WAI/ARIA/apg/patterns/textbox/)
- **Focusable:** yes
- **Keyboard:** Tab moves focus to the input; Standard text editing keys work natively
- **WCAG:** 1.3.1 Info and Relationships, 1.3.5 Identify Input Purpose, 1.4.3 Contrast (Minimum), 2.4.6 Headings and Labels, 3.3.1 Error Identification, 3.3.2 Labels or Instructions

**Rules**

- aria-errormessage points to error div; aria-describedby mirrors the link while invalid as AT fallback.
- When valid, aria-describedby points to helper text. When invalid, it switches to error text.
- delegatesFocus ensures host element forwards focus to inner <input>.
- ElementInternals.setFormValue() keeps the form value in sync with the input.
