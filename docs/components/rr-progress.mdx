| `indeterminate` | `boolean` | `false` | Show an animated indeterminate state instead of a fixed value. Reflected to an attribute. |

## Accessibility

- **WAI-ARIA pattern:** [www.w3.org/WAI/ARIA/apg/](https://www.w3.org/WAI/ARIA/apg/)
- **Focusable:** no
- **WCAG:** 1.4.3 Contrast (Minimum), 4.1.2 Name, Role, Value

**Rules**

- aria-valuenow is omitted and aria-busy="true" is set in indeterminate mode
- aria-valuemin is always 0 and aria-valuemax tracks the max prop
