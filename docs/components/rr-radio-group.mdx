## Overview

Use `rr-radio-group` for choosing exactly one of 2–5 visible, mutually exclusive options — plan tiers, shipping methods, yes/no/maybe. Above ~5 options, or when space is tight, reach for `rr-select` instead; when choices are independent, use checkboxes. The group is the form-associated element (fieldset/legend semantics, one `name`, one submitted `value`) — individual `rr-radio` children just declare their `value` and never wire their own form state. `orientation="horizontal"` fits short labels in a row; default vertical scans better for anything longer.

## Examples

### Default vertical group

```html
<rr-radio-group name="size" label="Size" value="md">
  <rr-radio value="sm">Small</rr-radio>
  <rr-radio value="md">Medium</rr-radio>
  <rr-radio value="lg">Large</rr-radio>
</rr-radio-group>
```

### Horizontal orientation

```html
<rr-radio-group name="size" label="Size" orientation="horizontal">
  <rr-radio value="sm">Small</rr-radio>
  <rr-radio value="md">Medium</rr-radio>
</rr-radio-group>
```

### Error state

```html
<rr-radio-group name="size" label="Size" required error-text="Please select a size">
  <rr-radio value="sm">Small</rr-radio>
  <rr-radio value="md">Medium</rr-radio>
</rr-radio-group>
```

### Disabled group

```html
<rr-radio-group name="size" label="Size" disabled>
  <rr-radio value="sm">Small</rr-radio>
  <rr-radio value="md">Medium</rr-radio>
</rr-radio-group>
```

## Properties

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | `` | Form field name submitted with the selected value. |
| `value` | `string` | `` | Selected radio value. Syncs the checked state of child radios. |
| `label` | `string` | `` | Group label rendered as the legend. |
| `errorText` | `string` | `` | Error message; sets data-invalid and a custom validity. Maps to attribute error-text. |
| `required` | `boolean` | `false` | Marks the group as required. |
| `disabled` | `boolean` | `false` | Disables the group and all child radios. |
| `orientation` | `'vertical' \| 'horizontal'` | `vertical` | Layout direction of the radio options. |
