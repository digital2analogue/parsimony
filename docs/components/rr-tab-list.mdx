## Accessibility

- **WAI-ARIA pattern:** [www.w3.org/WAI/ARIA/apg/patterns/tabs/](https://www.w3.org/WAI/ARIA/apg/patterns/tabs/)
- **Focusable:** no
- **Keyboard:** ArrowRight moves selection to the next enabled tab (wraps); ArrowLeft moves selection to the previous enabled tab (wraps); Home selects the first enabled tab; End selects the last enabled tab
- **WCAG:** 1.4.3 Contrast (Minimum), 2.1.1 Keyboard, 2.4.7 Focus Visible, 4.1.2 Name, Role, Value

**Rules**

- Implements roving tabindex across child rr-tab elements; disabled tabs are skipped during arrow navigation.
- label is required so the tablist has an accessible name via aria-label.
